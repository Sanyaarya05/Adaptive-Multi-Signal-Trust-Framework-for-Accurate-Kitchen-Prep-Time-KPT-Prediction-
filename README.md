# Adaptive-Multi-Signal-Trust-Framework-for-Accurate-Kitchen-Prep-Time-KPT-Prediction-
Adaptive KPT Prediction Framework that corrects noisy merchant signals using rider arrival data and error-based feedback loops. Detects and fixes marking biases (LPEM &amp; EPLM) via confidence-weighted overrides. Includes a gamified merchant trust-scoring system to improve signal reliability at scale.
